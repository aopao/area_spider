# Desc

基于Python Scrapy 的中国五级行政区划爬虫（数据来自国家统计局）。

# Prerequisites

- [Python](https://www.python.org/) 3.x
- [Scrapy](https://scrapy.org/) 1.5.x
- [pandas](http://pandas.pydata.org/) 0.20.x

# Run

```shell
$ scrapy crawl area_spider
```
